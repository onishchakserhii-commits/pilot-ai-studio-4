# Hospitality Website Audit

Use this skill when auditing a website for a restaurant or hotel, especially in the Montreux/Riviera area, to propose AI solutions.

## Step-by-Step Instructions
1. **Initial Assessment:** Проаналізуй головну сторінку. Визнач Value Proposition. Чи зрозуміло клієнту (гостю/відвідувачу) основну послугу за 5 секунд?
2. **Industry-Specific Check:** 
   - **Для ресторанів:** Чи легко знайти меню, бронювання столика та актуальний графік роботи?
   - **Для готелів:** Чи є зрозуміла форма бронювання, інформація про локацію та зручності (SPA, сніданки)?
3. **SEO & Localization:** Перевір наявність ключових слів (Montreux, Riviera). Чи оптимізовані сторінки під пошук "ресторан у Монтре" або "готель біля озера"?
4. **UX & Conversion:** Оціни мобільну адаптивність (критично для туристів!). Чи є чіткі кнопки дії (CTA)?
5. **AI Integration Potential (Industry Focus):** Запропонуй 3 точки інтеграції:
   - **Ресторани:** AI-чат для бронювання столиків, авто-відповіді на Google/TripAdvisor відгуки, динамічне меню.
   - **Готелі:** AI-консьєрж (відповіді на питання про check-in/check-out, поради щодо активностей у Монтре), автоматизація email-підтверджень.
   - **Обидва:** Аналіз клієнтських вподобань для персоналізованих пропозицій.
6. **Output Formatting:** Сформуй відповідь:
   - **Резюме аудиту:** (стисло).
   - **Таблиця можливостей:**
     | Сфера | Проблема | AI-рішення | Пріоритет |
     |-------|----------|------------|-----------|
     | ...   | ...      | ...        | High/Low  |
   - **Пропозиція:** Пітч (2-3 речення), орієнтований на Pilot AI Studio (як ми зробимо їхній бізнес сучаснішим за конкурентів у Riviera).
